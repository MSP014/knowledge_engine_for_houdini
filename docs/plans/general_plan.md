# Этап Действие Результат

1. Фундамент
    Задача: Создание структуры в папке проекта, настройка .gitignore и README.md.
    Результат: Чистый репозиторий, готовый к Open Source.
2. Экстрактор
    Задача: Написание парсера для houdini/help. Конвертация в Markdown с сохранением метаданных (билд, контекст).
    Результат: Набор нормализованных данных, готовых к индексации.
3. Склад (Qdrant)
    Задача: Написание docker-compose.yml и создание коллекции в Qdrant с Payload-схемой.
    Результат: Работающая векторная база на локалхосте или Synology NAS.
4. Мозги (RAG)
    Задача: Реализация логики поиска с Evidence Protocol (цитирование путей и заголовков).
    Результат: Агент, который не галлюцинирует, а «тыкает пальцем» в справку.
5. Интерфейс
    Задача: Создание Python Panel (PySide6) для Houdini.
    Результат: Окно поиска прямо в интерфейсе рядом со Scene View.
6. Мастер
    Задача: Скрипт-инсталлер для выбора режима (Local / NAS) и Docker-сборка.
    Результат: Готовый продукт для GitHub, который ставится одной командой.
